# Lab guide

## Lab 1 — environment basics

Reads `VEHICLE_NAME` and verifies that the development environment can execute project code.

## Lab 2 — ROS and odometry

Introduces publishers, subscribers, camera conversion, wheel commands, encoder data, differential-drive odometry, and trajectory plots.

## Lab 3 — vision and controllers

Builds camera distortion correction, image preprocessing, colored-line detection, P/PD/PID controllers, lane following, and a state machine as separate exercises.

## Final lab — integrated navigation

Turns the separate concepts into one installable ROS package with a documented pipeline, typed detection interface, safe state machine, and diagnostic launchers.

Each lab preserves the educational context of the previous one. Use the final lab for the maintained runtime.
